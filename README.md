# Encurtador de URL em Java

## Descrição
Este projeto é um encurtador de URL desenvolvido em Java. Ele permite que os usuários encurtem URLs longas para versões mais curtas e fáceis de compartilhar.


## Funcionalidades
- **Encurtar URL**: Permite que os usuários encurtem URLs longas.
- **Redirecionamento**: Redireciona para a URL original quando a URL curta é acessada.

## Tecnologias Utilizadas
- **Java**: Linguagem de programação principal.
- **Spring Boot**: Framework para criação de aplicações Java.
- **Maven**: Ferramenta de gerenciamento de dependências e build.
